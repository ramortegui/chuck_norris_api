# ChuckNorris

OTP application to consume [https://api.chucknorris.io](https://api.chucknorris.io)

##

## Installation

The package can be installed by adding `chuck_norris` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:chuck_norris, "~> 0.1.0"}
  ]
end
```
